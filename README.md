# 🥊 Champions Boxing Gym - Website Template

A modern, fully-featured, and responsive website template for boxing gyms and fitness facilities. Built with HTML5, CSS3, and vanilla JavaScript for optimal performance and ease of customization.

## ✨ Features

### 🎨 Design & UX
- **Responsive Design**: Mobile-first approach optimized for all devices (mobile, tablet, desktop)
- **Dark Mode Toggle**: Built-in theme switcher with localStorage persistence
- **Smooth Animations**: Fade-in effects, hover transitions, and scroll animations
- **Professional Styling**: Modern gradient effects and card-based layouts
- **Fixed Navigation**: Sticky header with smooth scroll-to-section functionality
- **Mobile Menu**: Hamburger menu for better mobile navigation

### 📱 Responsive Features
- Fully mobile-optimized with breakpoints for tablets and phones
- Touch-friendly buttons and interactive elements
- Optimized images and lazy loading support
- Flexible grid layouts that adapt to screen size

### 🏠 Sections Included
1. **Hero Section**: Eye-catching banner with call-to-action buttons
2. **About Section**: Gym overview with key features
3. **Statistics**: Impressive metrics showcase
4. **Training Programs**: Multiple program cards with descriptions
5. **Testimonials**: Member reviews with star ratings
6. **Pricing Plans**: Three-tier membership options with featured plan
7. **Weekly Schedule**: Comprehensive class schedule table
8. **Gallery**: Image gallery with hover effects
9. **Contact Form**: Integrated WhatsApp form with validation
10. **Footer**: Multi-column footer with social links

### 🔧 Technical Features
- **Form Validation**: Client-side validation with error messages
- **WhatsApp Integration**: Direct messaging via WhatsApp Business API
- **Smooth Scrolling**: Native scroll behavior with offset
- **Scroll-to-Top Button**: Fixed button for easy navigation
- **Performance Optimized**: Minimal dependencies, fast load times
- **SEO Ready**: Semantic HTML structure with proper headings

### 🎯 Interactive Elements
- Hover animations on cards and buttons
- Gallery overlay effects
- Form validation and error handling
- Mobile menu toggle
- Theme switcher
- Smooth page transitions
